[Collab.Land ](https://www.collab.land/)es un bot para la gestión y participación de comunidades que puede ser agregado a comunidades de Discord o Telegram.   

Sus funcionalidades principales son:

- Verificar si un usuario posee un determinado activo digital (token ERC) necesario para acceder a un canal específico o tener un cierto rol, y en caso afirmativo, otorgarles acceso.- Expulsar automáticamente a un usuario de un canal o rol restringido si vende u obviamente se deshace del token requerido para estar en él.- Dar propinas (tips) a otros usuarios dentro de un servidor.**¿Cómo funciona?**

Collab.Land utiliza mensajes firmados para vincular una cuenta de Discord con una dirección de Ethereum, verificando que el usuario de Discord es el propietario de la cuenta que dicen ser.

Una vez que se verifica que un usuario posee el activo digital requerido para los roles y canales dentro de Discord, Collab.Land les otorga automáticamente acceso.
